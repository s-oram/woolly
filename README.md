# Woolly

A VS Code extension to jump between component files.

If you have project with files like:

- /Button.tsx
- /Button.module.scss
- /Button.stories.tsx
- /Button.test.tsx

Woolly provides a couple of commands to jump between these files
- Go To Primary: Opens `Button.tsx`
- Go To Styles: Opens `Button.module.scss`
- Go To Stories: Open `Button.stories.tsx`
- Go To Tests: Open `Button.test.tsx`
